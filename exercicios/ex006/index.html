Formatações em HTML (Semantico e Não Semantico)

Negrito

Para colocar um texto em negrito, podemos utilizar a tag B. Porém, esta tag não é semantica.

Porém tem outra maneira de colocar um texto em negrito, utilizando a tag STRONG, que já é semantica.

Itálico

Para destacar um texto em itálico, podemos utilizar a tag I, que não é semântica.

Mas também podemos usar a tag EM, que é semantica.

Marcação

Para marcar um texto, dando-lhe destaque, podemos utilizar a tag MARK.

Para mudar a cor da marcação, podemos utilizar a parâmetro style na tag <mark>, com o valor background-color: lime; por exemplo.

Texto Grande

Para colocar um texto ligeiramente maior que o restante, dá-se uso à tag BIG. Porém esta tag é Obsoleta e sem semantica.

Texto Pequeno

Para fazer o contrário do texto acima, ou seja colocar um texto ligeiramente menor que o outro, utilizamos a tag SMALL.

Rasurado

Para rasurar um texto, usa-se a tag DEL.

Inserido ou Sublinhado

Para sublinhar um texto, podemos utilizar a tag U, porém esta tag não é semantica

Ao invés disso, utilizamos a tag INS.

Sobrescrito

Para sobrescrever um texto, como por exemplo x2 + y3 = 13, usamos a tag SUP.

Subscrito

Para subscrever um texto, como por exemplo H2O, usamos a tag SUB.
