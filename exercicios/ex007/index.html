Código-Fonte

Para exibir um código, ou um texto monoespacial, podemos utilizar a tag CODE.
Por exemplo: document.getElementById('aboutme')

Citações

Para citar uma frase simples, podemos utilizar a tag Q.
Por exemplo: O amor é como a guerra: muito fácil de começar e muito díficil de terminar.

Para citar uma frase ou um bloco de texto (contém quebras de linha, bloco, etc...), utilizamos a taq BLOCKQUOTE.
Por exemplo:

A guerra, a princípio, é a esperança de que a gente vai se dar bem; em seguida, é a expectativa de que o outro vai se ferrar; depois, a satisfação de ver que o outro não se deu bem; e finalmente, a surpresa de ver que todo mundo se ferrou.

Abreviações

Abreviações em HTML servem para mostrar o significado de uma sigla ou abreviação. Para isso dá-se uso à tag ABBR.

Passe o ponteiro do mouse na palavra seguinte: HTML

Texto Invertido

Para inverter o texto verticalmente, utilizamos a tag BDO.

Por exemplo, Olá, este texto está invertido!
