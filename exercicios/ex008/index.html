Listas e Tipos de Listas

Listas Ordenadas

Para criar uma lista ordenada (que tem ordem numérica) existe a tag OL.
Dentro de cada lista existem os seus itens, que para isso se utiliza LI

  1. Primeiro Item
  2. Segundo Item
  3. Terceiro Item
  4. Quarto Item
  5. Quinto Item

Por padrão, a ordem numérica é identificada por numeração àrabe (1,2,3,4...), mas para mudar para outro tipo de ordem, utilizamos o parâmetro type="I" (Numeração Romana), type="A" (Ordem com letras maiusculas), type="a" (Ordem com letras minusculas).

Listas Não Ordenadas

Para criar uma lista não ordenada (que não tem ordem numérica) existe a tag UL.
Dentro de cada lista existem os seus itens, que para isso se utiliza LI

  • Primeiro Item
  • Segundo Item
  • Terceiro Item
  • Quarto Item
  • Quinto Item

Por padrão, a ordem é identificada por um circulo preto , mas para mudar para outro tipo de ordem, utilizamos o parâmetro type="disc" (Circulo preto padrão), type="circle" (Circulo Branco), type="square" (Quadrado Preto).

Listas de Definição

Para criar uma lista de definição (que define algo), usa-se a tag DL.
Dentro de cada item da lista existe o Termo (DT) e a Descrição (DD)

Termo
Descrição

Por exemplo:

CM
Correio da Manhã
JN
Jornal de Noticias
DN
Diario de Noticias

Misturar Listas Ordenadas e Não Ordenadas

Também podemos misturar os 3 tipos de listas. Basta colocar as tags OL, UL, ou DL, dentro da Lista. Por exemplo:

        
            <ol>
                <ul>
                    <li>Primeiro Item</li>
                    <li>Segundo Item</li>
                    <li>Terceiro Item</li>
                </ul>
            </ol>
  1. Listas Ordenadas:
    1. Primeiro item da lista ordenada
    2. Segundo item da lista ordenada
  2. Listas Não-Ordenadas
    • Primeiro item da lista não ordenada
    • Segundo item da lista não ordenada
  3. Listas de Definição
    4. Primeiro termo
    Primeira descrição
    Segundo termo
    Segunda descrição
